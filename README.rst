=========
Flamaster
=========

-------------------------------
E-commerce framework with Flask
-------------------------------

Setup requirements:

- python == v2.7.x
- pip >= 1.1
- flask >= 0.8
- flask-script >= 0.3.1 (for shell commands support)
- flask-sqlalchemy >= 0.15
- SQLAlchemy >= 0.7.5

Developer notes
_______________

We use CommonJS AMDnotation for JS modules definition
